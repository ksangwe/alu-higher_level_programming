# Python Object Relational Mapping

This project introduces Object Relational Mapping (ORM) using Python and SQLAlchemy.

## Description

In this project, we:
- Connect Python to a MySQL database
- Map Python classes to database tables
- Use SQLAlchemy to handle database operations

## Files

- `model_state.py`: Defines the State class mapped to the `states` table
- `6-model_state.py`: Script to create the table in MySQL

## Requirements

- Python 3
- MySQL
- SQLAlchemy

# Python - Object Relational Mapping

This project is part of the ALU Higher Level Programming curriculum.

## Description

This project connects Python with MySQL databases.

It covers:
- Using MySQLdb to execute SQL queries
- Using SQLAlchemy ORM to interact with databases using Python objects

## Learning Objectives

- Connect to a MySQL database from Python
- Execute SELECT and INSERT queries using Python
- Understand ORM (Object Relational Mapping)
- Map Python classes to database tables

## Requirements

- Python 3.8
- MySQL 8.0
- MySQLdb (mysqlclient)
- SQLAlchemy 1.4
