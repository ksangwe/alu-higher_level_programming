# Python Object Relational Mapping

This project introduces Object Relational Mapping (ORM) using Python and SQLAlchemy.

## Description

In this project, we:
- Connect Python to a MySQL database
- Map Python classes to database tables
- Use SQLAlchemy to handle database operations

## Files

- `model_state.py`: Defines the State class mapped to the `states` table
- `6-model_state.py`: Script to create the table in MySQL

## Requirements

- Python 3
- MySQL
- SQLAlchemy
