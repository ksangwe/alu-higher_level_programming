# Python - Object Relational Mapping

This project is part of the ALU Higher Level Programming curriculum.

## Description

This project connects Python with MySQL databases.

It covers:
- Using MySQLdb to execute SQL queries
- Using SQLAlchemy ORM to interact with databases using Python objects

## Learning Objectives

- Connect to a MySQL database from Python
- Execute SELECT and INSERT queries using Python
- Understand ORM (Object Relational Mapping)
- Map Python classes to database tables

## Requirements

- Python 3.8
- MySQL 8.0
- MySQLdb (mysqlclient)
- SQLAlchemy 1.4
