# SQL More Queries

This project focuses on advanced SQL concepts using MySQL 8.0.

## Topics Covered

- User creation and privileges (GRANT, REVOKE)
- Constraints (PRIMARY KEY, FOREIGN KEY, NOT NULL, UNIQUE)
- Subqueries
- Joins (INNER, LEFT, RIGHT)
- Multiple joins and DISTINCT
- UNION operations

## Requirements

- All SQL keywords must be in uppercase
- Each SQL file must start with a comment
- All files must end with a new line
